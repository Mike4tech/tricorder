This directory is a refactor of the python plugins so that they work
from the command line given a url.

So something like:

python wiley.py http://onlinelibrary.wiley.com/doi/10.1111/j.1469-8749.1975.tb04699.x/abstract?sms_ss=citeulike&at_xt=4d5ab3a1f2298daf%2C0

or

python plos.py http://www.plosone.org/article/info%3Adoi%2F10.1371%2Fjournal.pone.0065382
